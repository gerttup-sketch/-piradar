# Õpiradar

Õpiradar on tudengitele mõeldud veebirakendus, mis aitab hallata õppetöid, tähtaegu, prioriteete ja ülesannete staatust.

## Eesmärk
Rakenduse eesmärk on aidata tudengil saada parem ülevaade oma koolitöödest ja vältida tähtaegade ununemist.

## Peamised funktsioonid
- ülesannete lisamine
- aine ja tähtaja määramine
- prioriteedi ja staatuse valik
- otsing
- erinevad vaated
- teemade vahetamine

## Tehniline lahendus
Rakendus on loodud vaibkoodimise tööriista abil ning kasutab andmete hoidmiseks local storage'it.
